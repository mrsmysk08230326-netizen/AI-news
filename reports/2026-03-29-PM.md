## Claude Code 便利機能・実用法

### 2026年3月 大型アップデートまとめ

Claude Code はバージョン 2.1.63 → 2.1.76 へと更新され、多数の新機能が追加された。

#### 🎙️ 音声モード（Voice Mode）
- `/voice` コマンドで起動。スペースキーを押しながら話し、離すと送信するプッシュトゥトーク方式。
- 常時リスニングではなく、意図的・精密な操作が可能。
- 3月に10言語追加、計20言語の音声認識に対応。

#### 🖥️ コンピュータ使用（Computer Use）
- 3月23日、Pro・Max ユーザー向けにリリース。
- Claude がファイルを開き、開発ツールを起動し、クリック・スクロールなどデスクトップ操作を自動実行。
- スマートフォンの Claude アプリからリモートで指示可能。コード・ファイルはローカルに残り、チャットメッセージのみ暗号化送信。

#### 🔁 スケジュールタスク（`/loop`）
- 定期的なソフトウェア作業（テスト実行、ビルドチェックなど）を自動化できる繰り返しタスク機能。
- Auto モード・リモートコントロールと組み合わせると特に強力。

#### 🧠 拡張思考（Extended Thinking）& 適応的推論
- デフォルトで有効。`Ctrl+O` で Verbose モードをオンにすると推論過程を視覚的に確認できる。
- Opus 4.6 / Sonnet 4.6 は「effort level」設定に応じて思考トークン予算を動的に配分する適応的推論をサポート。

#### 📋 プランモード（Plan Mode）
- `Shift+Tab` で切り替え。読み取り専用のコードベース分析を行い、複雑な変更の計画をまとめてから実装に進める。

#### 🔧 MCP / ツール検索の改善
- **Lazy Loading**：必要なツールだけを動的に活性化し、起動オーバーヘッドを削減。
- MCP サーバーがインタラクティブフォームやブラウザを開いてデータ収集できるようになった。
- `Elicitation` / `ElicitationResult` フックで応答をインターセプト・上書き可能。

#### 📊 リッチ出力・ビジュアライゼーション
- インタラクティブなチャート・図・モバイルアプリの生成が可能に。

#### 💡 コンテキスト管理の改善
- `/context` コマンドがトークン消費の多いツールや記憶肥大化を特定し、最適化のヒントを提案。
- `autoMemoryDirectory` 設定で記憶ファイルの保存先をカスタマイズ可能。
- 記憶ファイルに最終更新タイムスタンプが付与された。

## 画像生成AI

### 主要アップデート

#### OpenAI — GPT-4o ネイティブ画像生成
- ChatGPT および Sora に統合。Plus・Pro・Team・Free 全ユーザーへロールアウト済み。
- テキスト生成と同じ GPT-4o モデルに内蔵された自己回帰型モデル。テキスト＆画像プロンプト入力対応。
- Artificial Analysis 画像アリーナで **Recraft V3・FLUX 1.1 Pro・Gemini 2.0 Flash** と並び同率1位を記録。
- 開発者 API も公開。顔・ロゴ・細部の編集精度が大幅向上。([OpenAI 発表](https://x.com/OpenAI/status/1904602845221187829) / [TechCrunch](https://techcrunch.com/2025/04/23/openai-makes-its-upgraded-image-generator-available-to-developers/))

#### Google — Nano Banana 2（2026年2月26日）
- Gemini 3.1 Flash Image の通称。Pro モデルの高忠実度を維持しつつ高速化。
- 512px〜4K の複数アスペクト比をサポート。全出力に **SynthID 透かし**を自動付与。([TechCrunch](https://techcrunch.com/2026/02/26/google-launches-nano-banana-2-model-with-faster-image-generation/))

- 米国の Google AI Ultra サブスクライバー向けに公開。2D 画像やプロンプトから探索可能な 3D ワールドを生成。([TechCrunch](https://techcrunch.com/2026/01/29/i-built-marshmallow-castles-in-googles-new-ai-world-generator-project-genie/))

#### Adobe Firefly — Photoshop AI アシスタント（2026年3月10日）

- Generative Fill・Generative Remove・Generative Expand・Generative Upscale を追加。([TechCrunch](https://techcrunch.com/2026/03/10/adobe-is-debuting-an-ai-assistant-for-photoshop/))

#### Gamma — Gamma Imagine（2026年3月17日）
- プレゼン作成プラットフォーム Gamma がブランド特化型画像生成ツールを追加。
- テキストプロンプトでマーケティング素材・ソーシャルグラフィック・インフォグラフィックを生成。
- ChatGPT・Claude・Make・Zapier・Atlassian 等と統合。([TechCrunch](https://techcrunch.com/2026/03/17/gamma-adds-ai-image-generation-tools-in-bid-to-take-on-canva-and-adobe/))

#### Meta — "Mango"（2026年上半期リリース予定）
- 画像＆動画生成モデル "Mango" と テキストモデル "Avocado" を上半期中にリリース予定。([TechCrunch](https://techcrunch.com/2025/12/19/meta-is-developing-a-new-image-and-video-model-for-a-2026-release-report-says/))

## 動画生成AI

### 主要アップデート

#### ByteDance — Dreamina Seedance 2.0（2026年3月26日）⭐ 最注目
- CapCut に正式統合。プロンプト・画像・参照動画から動画と音声を一括生成・編集・同期。
- ブラジル・インドネシア・マレーシア・メキシコなどから段階展開。全出力に透かし付与。
- **著作権問題**：Paramount がハリウッドの有名 IP を無断再現しているとして cease-and-desist を送付。([TechCrunch](https://techcrunch.com/2026/03/26/bytedances-new-ai-video-generation-model-dreamina-seedance-2-0-comes-to-capcut/))

#### Runway — $315M 調達 & Gen 4.5（2026年2月10日）
- Series E で $315M を調達、バリュエーション **$5.3B**。次世代ワールドモデルの事前学習に投資。
- **Gen 4.5**：テキストプロンプトから HD 動画を生成。ネイティブ音声・長編・マルチショット・キャラクター一貫性・高度編集ツールを搭載。
- Runway プラットフォーム内で Kling 3.0・Kling 2.6 Pro・WAN2.2 Animate・GPT-Image-1.5・Sora 2 Pro など他社モデルも利用可能に。([TechCrunch](https://techcrunch.com/2026/02/10/ai-video-startup-runway-raises-315m-at-5-3b-valuation-eyes-more-capable-world-models/))

#### Google — Veo 3 / Veo 3.1
- Artificial Analysis Video Arena で **テキスト→動画・画像→動画の両部門で1位**を獲得（Kling 2.0・Runway Gen 4 を超える）。
- Google Cloud Vertex AI Studio・Flow・米国 Gemini Advanced Ultra サブスクライバー向けに提供。
- 56研究者・32大学の推論ベンチマークでは **Veo 3.1: 48.0%**（Sora 2 の 54.6% に次ぐ2位）。

#### xAI — Grok Imagine 1.0（2026年2月1日）
- 720p・10秒動画生成、音声改善、プロンプト追従性向上。
- Artificial Analysis Video Arena でテキスト→動画・画像→動画の両部門で **Runway Gen-4.5・Kling 2.5 Turbo・Veo 3.1 を上回り1位**（xAI 発表時点）。

#### Kling（Kuaishou）
- Kling 3.0 が最新版。JPMorgan が年間収益 **$2.4億** 見込みで「世界で最も過小評価された AI 株のひとつ」と評価。
- Google の Veo・OpenAI の Sora と真正面から競合。

#### Adobe Firefly — Quick Cut（2026年2月25日）
- 自然言語の指示でフッテージの不要部分を自動カットし、つなぎ合わせて初稿を作成。
- プロジェクト全体・特定タイムライン・選択クリップ単位で適用可能。([TechCrunch](https://techcrunch.com/2026/02/25/adobe-fireflys-video-editor-can-now-automatically-create-a-first-draft-from-footage/))

#### 推論ベンチマーク（56研究者・32大学）
研究者らが「AI 動画業界は視覚品質だけを最適化し、推論能力を軽視している」と指摘。スコア：Sora 2（54.6%）> Veo 3.1（48.0%）> Runway Gen-4 Turbo（40.3%）> Kling 2.6（36.9%）（人間基準: 97.4%）。
